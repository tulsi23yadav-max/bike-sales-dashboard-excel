# Bike Sales Dashboard (Excel Project)

## Project Overview
This project analyzes bike sales data using Microsoft Excel.

## Skills Used
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Excel Functions

## Key Insights
- Analyzed income vs bike purchase
- Compared age brackets
- Studied customer commute patterns
- Filtered data using marital status slicer
